*  `apt update`
*  `apt install git -y`
*  `git clone git: // github.com / 3l-tech / nexphisher.git`
*  `cd nexphisher`
####  > CONFIGURAÇÃO: `configuração do bash`
####  > CONFIGURAÇÃO [TERMUX]: `bash tmux_setup`
